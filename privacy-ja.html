Sleep now プライバシーポリシー

最終更新日: 2026年7月3日
発効日: 2026年7月3日

Rio Naruse(以下「提供者」といいます)は、提供者が提供する自然音再生・睡眠支援アプリケーション「Sleep now」(以下「本アプリ」といいます)におけるユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー(以下「本ポリシー」といいます)を定めます。

第1条(収集する情報)

提供者は、本アプリの提供にあたり、以下の情報を収集する場合があります。

1. ユーザーが提供する情報

Sign in with Apple によるログインの場合: - Apple ID に紐づくメールアドレス(実メールアドレス、または Apple が提供するリレーメールアドレス) - 氏名(ユーザーが提供に同意した場合のみ) - 一意のユーザー識別子(user identifier)

Google アカウントによるログインの場合: - Google アカウントのメールアドレス - 氏名 - プロフィール画像 URL(取得する場合) - 一意のユーザー識別子

ゲスト(匿名)ログインの場合: - Firebase Authentication が生成する一意の匿名 ID のみ - 個人を特定できる情報は収集しません

2. 自動的に収集される情報

  • デバイス情報: OS バージョン、デバイスモデル、言語設定
  • アプリ使用情報: 起動日時、使用機能、クラッシュログ(Firebase Crashlytics 等を導入する場合)
  • タイムゾーン情報

3. 収集しない情報

本アプリは以下の情報を収集しません: - 位置情報(GPS) - 連絡先 - カメラ・マイクへのアクセス - 写真ライブラリへのアクセス - ヘルスケアデータ - 広告識別子(IDFA)

第2条(情報の利用目的)

提供者は、収集した情報を以下の目的で利用します。

  1. 本アプリの提供、維持、改善のため
  2. ユーザー認証および不正利用防止のため
  3. ユーザーからのお問い合わせへの対応のため
  4. 本アプリに関する重要なお知らせをお届けするため
  5. 有料機能(プレミアム機能)の提供および課金処理のため
  6. 統計データの作成(個人を特定できない形式)のため
  7. 利用規約に違反する行為への対応のため
  8. その他、上記の利用目的に付随する目的のため

第3条(第三者への情報提供)

提供者は、以下の場合を除き、ユーザーの同意を得ずに第三者に個人情報を提供することはありません。

  1. 法令に基づく場合
  2. 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  3. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
  4. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

第4条(利用する外部サービス)

本アプリは、以下の外部サービスを利用しています。各サービスにおける個人情報の取扱いについては、各サービスのプライバシーポリシーをご確認ください。

1. Firebase(Google LLC)

  • 利用サービス: Firebase Authentication, Firebase Firestore(利用する場合)
  • 収集情報: ユーザー認証情報、匿名ユーザー ID
  • プライバシーポリシー: https://policies.google.com/privacy

2. Sign in with Apple(Apple Inc.)

  • 収集情報: Apple ID に紐づく情報(ユーザーの同意に基づく)
  • プライバシーポリシー: https://www.apple.com/legal/privacy/

3. Google Sign-In(Google LLC)

  • 収集情報: Google アカウント情報
  • プライバシーポリシー: https://policies.google.com/privacy

4. Apple App Store(Apple Inc.)

  • 利用目的: 課金処理、アプリ配信
  • プライバシーポリシー: https://www.apple.com/legal/privacy/

第5条(データの保存場所)

提供者は、収集した個人情報を Firebase(Google LLC)が提供するサーバーに保存します。Firebase のサーバーは、主にアメリカ合衆国等に所在する可能性があります。

Firebase のデータリージョンについては、Google のセキュリティおよびプライバシーポリシーに従い、適切に管理されます。

第6条(データの保存期間)

  1. 提供者は、ユーザーの個人情報を、本ポリシーに定める利用目的の達成に必要な範囲において保存します。
  2. アカウント削除依頼を受けた場合、提供者は合理的な期間内(通常30日以内)に該当する個人情報を削除します。
  3. 法令上の保存義務がある情報については、法令に定める期間保存します。
  4. ゲスト(匿名)アカウントは、Firebase Authentication のポリシーに従って管理されます。

第7条(ユーザーの権利)

ユーザーは、以下の権利を有します。

1. アクセス権

自己の個人情報にアクセスし、その内容を確認する権利。

2. 訂正権

自己の個人情報が不正確または不完全な場合、訂正を求める権利。

3. 削除権(忘れられる権利)

自己の個人情報の削除を求める権利。

4. 処理制限権

自己の個人情報の処理の制限を求める権利。

5. データポータビリティ権

自己の個人情報を構造化された、一般的で機械可読な形式で受け取る権利。

6. 異議申立権

自己の個人情報の処理に対して異議を申し立てる権利。

これらの権利を行使する場合は、steplife.contact@gmail.com までご連絡ください。

本アプリは、13歳未満のお子様を対象としていません。提供者は、13歳未満のお子様から意図的に個人情報を収集することはありません。13歳未満のお子様が本アプリを利用していることが判明した場合、速やかに該当情報を削除いたします。

保護者の方で、お子様が本アプリを利用していることに気づかれた場合は、steplife.contact@gmail.com までご連絡ください。

  1. 監督機関への異議申立権: EU/UK の各データ保護当局に対して異議を申し立てる権利
  2. 同意の撤回権: 個人情報の処理について同意を撤回する権利

第10条(米国カリフォルニア州居住者の権利 - CCPA/CPRA)

カリフォルニア州にお住まいのユーザーは、カリフォルニア州消費者プライバシー法(CCPA)および California Privacy Rights Act(CPRA)に基づき、以下の権利を有します。

  1. 収集された個人情報のカテゴリおよび具体的な内容を知る権利
  2. 個人情報の削除を求める権利
  3. 個人情報の販売または共有に対するオプトアウト権(提供者はユーザーの個人情報を販売しません)
  4. これらの権利を行使したことによる差別を受けない権利

第11条(Cookie およびトラッキング技術)

本アプリはネイティブモバイルアプリのため、Web の Cookie は使用しません。ただし、Firebase 等の SDK が識別子を保存する場合があります。これらの識別子は、ユーザー認証および本アプリの機能提供のために使用されます。

本アプリは、広告のためのトラッキング(IDFA)を使用しません。

第12条(セキュリティ)

提供者は、個人情報の紛失、盗難、不正アクセス、開示、改ざん、破壊等を防止するため、合理的なセキュリティ対策を講じています。ただし、インターネットを介したデータ送信および電子的保存には絶対的な安全性はなく、提供者は完全な安全性を保証するものではありません。

具体的な対策: - Firebase による認証情報の暗号化 - HTTPS 通信によるデータの暗号化 - アクセス権限の適切な管理

第13条(本ポリシーの変更)

  1. 提供者は、必要と判断した場合には、ユーザーに事前の通知なくいつでも本ポリシーを変更することができるものとします。
  2. 重要な変更がある場合は、本アプリ内での通知またはメールにてお知らせします。
  3. 変更後の本ポリシーは、本アプリまたは提供者のウェブサイトに掲載された時点から効力を生じます。
