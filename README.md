# Sleep now — Legal Documents

Sleep now アプリの利用規約・プライバシーポリシー公開用リポジトリ。

## 公開URL

https://steplifecontact-rgb.github.io/sleep-now/

## ファイル構成

- `index.html` — インデックスページ
- `style.css` — 共通スタイル
- `terms-ja.html` / `terms-en.html` — 利用規約 (日/英)
- `privacy-ja.html` / `privacy-en.html` — プライバシーポリシー (日/英)
- `*.md` — 原文Markdown (バックアップ用)

## メンテナンス

内容を変更する場合:
1. `*.md` ファイルを編集
2. Python で HTML を再生成 (`python3 build.py` 等) or 直接 `.html` を編集
3. commit & push
